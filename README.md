# SeeSpace 🌌

Projeto de site educacional com foco em introdução à astronomia.

## 📖 Objetivo
Desenvolver uma aplicação web simples para apresentar conceitos básicos sobre astronomia de forma acessível.

## 🛠 Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## 🚀 Status
Projeto concluído como prática de desenvolvimento front-end.

## 📌 Observação
Projeto com finalidade educacional.
